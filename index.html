Welcome to Memory Game!

About the game:

This game is primarily built by Prem Dahal for a springBoard Bootcamp exercise. This is a simple game built using basic HTML, Css, and Javascript.

Instructions:

There are total of 10 rectangular boxes. As you click on any individual box, a background color will change to the color of that box. After you click on the second box, the background of those two boxes will be compared. if those color matches, it will stay up. If the color do not match, the box will reset and changes the background color to white. Reset button can be used at any point of time to reset the game.

Score

The minimum clicks needed to complete the game is 10. The score is 10/#of clicks . Your the number of guesses, higher the score. Your score will be displayed at the end of the game. Your previous game score is displayed on the game screen.
